# StackSmash

StackSmash is a simple, fast website that lets you test your typing speed using programming related words and cool visual effects. It's written using only HTML, CSS, and JS.

## Feature List:

- **Programming Wordlists**: Practice with words from popular programming languages and libraries.
- **Real-Time Stats**: See your WPM, accuracy, and error highlights as you type.
- **Responsive Design**: Works on desktop, maybe mobile.
