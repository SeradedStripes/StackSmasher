# StackSmash

StackSmash is a simple, fast website that lets you test your typing speed using programming related words and cool visual effects. It's written using only HTML, CSS, and JS.

## Feature List:

- **Programming Wordlists**: Practice with words from popular programming languages and libraries.
- **Real-Time Stats**: See your WPM, accuracy, and error highlights as you type.
- **Fun Visual Effects**: Enjoy animations and visual flourishes that make typing practice more engaging.
- **Custom Challenges**:
  - Disappearing Words: Words vanish after a certain time, adding an extra layer of challenge.
  - Time-Limited Modes: Race against time to see how many words you can type accurately.
  - Shakespeare Mode: Type classic literature excerpts for a fun twist on programming practice.
  - Code Mode: Type actual code snippets to improve your coding speed and accuracy.
- **Responsive Design**: Works on desktop, maybe mobile.
